# 혜택 설정 탭 UI 가이드

## 📋 개요

혜택 설정 탭은 게임별 리워드를 설정하고 관리하는 페이지입니다. 활성화된 게임에 대해서만 혜택 설정이 가능합니다.

---

## 🎨 UI 구조

### 1. 페이지 헤더
```
┌─────────────────────────────────────────┐
│ 혜택 설정                                │
│ 게임별 리워드를 설정하고 고객 만족도를   │
│ 높이세요.                                │
└─────────────────────────────────────────┘
```

**구성 요소**:
- **제목**: "혜택 설정" (28px, 굵게)
- **부제**: "게임별 리워드를 설정하고 고객 만족도를 높이세요." (15px, 회색)

---

### 2. 정보 박스
```
┌─────────────────────────────────────────┐
│ 💡 기본 설정: 게임별로 점수 구간을      │
│    설정하고 각 구간마다 제공할 쿠폰     │
│    혜택을 자유롭게 설정할 수 있습니다.  │
└─────────────────────────────────────────┘
```

**스타일**:
- 배경색: 연한 파란색 (#e7f1ff)
- 테두리: 1px solid #b6d4fe
- 아이콘: 전구 모양 (노란색)

---

### 3. 빈 상태 (Empty State)

활성화된 게임이 없을 때 표시:

```
┌─────────────────────────────────────────┐
│                                          │
│           🎮                             │
│                                          │
│     활성화된 게임이 없습니다             │
│                                          │
│ 게임 설정 탭에서 먼저 게임을 활성화해주세요│
│                                          │
│    [ 게임 설정으로 이동 ]                │
│                                          │
└─────────────────────────────────────────┘
```

**구성 요소**:
- 게임패드 아이콘 (100px, 파란색)
- 제목: "활성화된 게임이 없습니다"
- 설명: "게임 설정 탭에서 먼저 게임을 활성화해주세요"
- 버튼: "게임 설정으로 이동" (파란색 그라데이션)

---

### 4. 게임 카드 그리드

3열 그리드 레이아웃으로 활성화된 게임 카드를 표시:

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ 게임 1   │  │ 게임 2   │  │ 게임 3   │
│          │  │          │  │          │
│ [메달]   │  │ [메달]   │  │ [메달]   │
│          │  │          │  │          │
│ [버튼]   │  │ [버튼]   │  │ [버튼]   │
└──────────┘  └──────────┘  └──────────┘
```

---

### 5. 게임 카드 상세 구조

#### 카드 헤더
```
┌─────────────────────────────────────────┐
│ 🎯 브랜드 퀴즈           [▲ 접기]       │
└─────────────────────────────────────────┘
```

**구성 요소**:
- 게임 아이콘 (원형, 40px)
  - 브랜드 퀴즈: 🎯 (빨간색 배경)
  - 메뉴 픽 맞추기: 🍴 (보라색 배경)
  - 같은 카드 찾기: 📋 (파란색 배경)
  - 틀린 그림 찾기: 🔍 (파란색 배경)
- 게임 이름 (18px, 굵게)
- 접기/펼치기 버튼

#### 메달 요약 (항상 표시)
```
┌─────────────────────────────────────────┐
│ ┌──────┐  ┌──────┐  ┌──────┐           │
│ │  🥉  │  │  🥈  │  │  🥇  │           │
│ │동메달│  │은메달│  │금메달│           │
│ │6-7점 │  │8-9점 │  │10점 │           │
│ └──────┘  └──────┘  └──────┘           │
└─────────────────────────────────────────┘
```

**메달 박스**:
- 배경: 연한 회색 (#f8f9fa)
- 메달 아이콘 색상:
  - 동메달: #cd7f32
  - 은메달: #adb5bd
  - 금메달: #fab005

#### 카드 콘텐츠 (펼쳤을 때만 표시)

**템플릿 버튼**:
```
┌─────────────────────────────────────────┐
│        🪄 템플릿 적용                    │
└─────────────────────────────────────────┘
```

**상세 통계**:
```
┌─────────────────────────────────────────┐
│ 참여자       최고 점수    평균 플레이 시간│
│ 127명        10점         15초           │
└─────────────────────────────────────────┘
```

**액션 버튼**:
```
┌─────────────────────────────────────────┐
│ [ ➕ 단계 추가 ]           [ 저장 ]      │
└─────────────────────────────────────────┘
```

- 통계 그리드: 2열

## 🔧 인터랙션

### 1. 카드 접기/펼치기
- **접힌 상태**: 헤더 + 메달 요약만 표시
- **펼친 상태**: 전체 콘텐츠 표시
- 토글 버튼: "▼ 더보기" / "▲ 접기"

### 2. 템플릿 적용 버튼
- 클릭 시 → BenefitSettingModal 열림
- 모달에서 혜택 추가/편집 가능

### 3. 저장 버튼
- 클릭 시 → BenefitDetailsModal 열림
- 설정된 혜택 목록 확인 가능

### 4. 게임 설정으로 이동 버튼
- 클릭 시 → 게임 설정 탭으로 이동
- CustomEvent 'switch-tab' 발생

```
            └── Action Buttons

├── BenefitSettingModal (혜택 추가/편집)
└── BenefitDetailsModal (혜택 목록 확인)
```

## 🔗 연동 모달

### BenefitSettingModal
- **용도**: 새 혜택 추가 또는 템플릿 적용
- **트리거**: "템플릿 적용" 또는 "단계 추가" 버튼
- **Props**:
  - gameType: 게임 타입
  - gameName: 게임 이름
  - gameIcon: 게임 아이콘
  - qrCodeId: QR 코드 ID

### BenefitDetailsModal
- **용도**: 설정된 혜택 목록 확인 및 관리
- **트리거**: "저장" 버튼
- **Props**:
  - gameType: 게임 타입
  - gameName: 게임 이름
  - gameIcon: 게임 아이콘
  - qrCodeId: QR 코드 ID
  - stats: 게임 통계 데이터

## 📊 데이터 구조

### GameBenefit 인터페이스
```typescript
interface GameBenefit {
  type: string           // 게임 타입 (pinball, brick-breaker 등)
  name: string           // 게임 이름
  icon: string           // FontAwesome 클래스명
  enabled: boolean       // 활성화 여부
  stats: GameStats       // 게임 통계
}
```

### GameStats 인터페이스
```typescript
interface GameStats {
  todayPlays: number      // 오늘 플레이 횟수
  avgScore: number        // 평균 점수
  participants: number    // 참여자 수
  couponsIssued: number   // 발급된 쿠폰 수
  highestScore: number    // 최고 점수
  avgPlayTime: number     // 평균 플레이 시간 (초)
}
```

## ✨ 주요 기능

1. **동적 게임 로딩**: 게임 설정 탭에서 활성화한 게임만 표시
2. **실시간 통계**: 게임별 참여자, 점수, 플레이 시간 등 표시
3. **카드 접기/펼치기**: 공간 효율적인 UI
4. **모달 연동**: 혜택 설정/상세보기 모달과 연동
5. **탭 전환**: 게임 설정 탭으로 원클릭 이동
